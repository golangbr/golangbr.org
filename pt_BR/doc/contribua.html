Como você pode ajudar

Relatando problemas

Se você encontrar bugs, erros, ou inconsistências no código do projeto Go ou documentação, por favor, avise-nos registrando uma issue em nosso tracker de issues. (Claro, você deve verificar se não é um pedido existente antes de criar um novo.)

Nos orgulhamos de ser meticulosos; nenhum pedido é muito pequeno.

Contribuindo com o código

Go é um projeto open source e contribuições são bem vindas na comunidade.

Para começar, leia essas diretrizes de contribuição para obter informações sobre design, testes e nosso processo de revisão de código.

Verifique seguidor para abrir pedidos que lhe interessam. Os status HelpWanted são particularmente a necessidade de ajuda externa.
